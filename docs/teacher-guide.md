# 教師使用指南

## 建議教學流程（45–50 分鐘）

1. **地形總覽（8 分鐘）**：觀察衛星圖與山脈圖層，讓學生比較南北美洲西部高山帶與東部低地。
2. **山脈與河流（10 分鐘）**：同時開啟山脈、河流圖層，討論分水嶺與河流方向。
3. **特殊地形（12 分鐘）**：比較大峽谷、黃石地熱區、猶加敦天然井及莫哈維沙漠。
4. **國家公園案例（10 分鐘）**：每組選一座國家公園，閱讀資訊卡並開啟官方來源。
5. **隨機挑戰（5–10 分鐘）**：全班共同作答或分組競賽。

## 課堂任務範例

- 找出一條流向北冰洋的河流。
- 比較洛磯山脈與阿帕拉契山脈的外觀差異。
- 找出兩座具有冰河地形的國家公園。
- 說明大峽谷與科羅拉多河的因果關係。
- 解釋莫哈維沙漠與山脈雨影效應的關係。

## 資訊素養提醒

請學生區分「地圖上的概略示意」與「原始科學資料」，並要求學生在報告中保留官方來源名稱與連結。
